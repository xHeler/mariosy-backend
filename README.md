## 🔨 Running guide
Before you run project install and configure java with environment variable.

###  🟢 Running project
```bash
./mvnw exec:java -Dexec.mainClass="com.deloitte.ads.MariosyApplication"
```

###  🐳 Running database
```bash
docker compose up -d
```
